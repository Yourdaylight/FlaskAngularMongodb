# 二、通用接口（登录/注册）
## 1 登录、注册
### 1.1 登录
post /login
#### 参数
| 参数名       | 类型 | 说明 |
|:----------| :--- | :--- |
|  username | string | 用户名|
|  password | string | 密码|

#### 返回值
| 参数名       | 类型 | 说明          |
|:----------| :--- |:------------|
|  code | int | 状态码,0成功1失败. |
|  msg | string | 提示信息        |
|  data | object | 返回数据        |

### 1.2 注册
post /register
#### 参数
| 参数名       | 类型 | 说明 |
|:----------| :--- | :--- |
|  username | string | 用户名|
|  password | string | 密码|

#### 返回值
| 参数名       | 类型 | 说明          |
|:----------| :--- |:------------|
|  code | int | 状态码,0成功1失败. |
|  msg | string | 提示信息        |
|  data | object | 返回数据        |
